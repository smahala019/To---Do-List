# 📋 To-Do Pro

A modern, gamified productivity web app that helps you manage daily tasks, track progress, earn XP, and stay consistent with your goals.

Built with HTML, CSS, and JavaScript, this lightweight app runs entirely in the browser with no backend required.

---

## ✨ Features

✔ Modern UI with Dark / Light theme
✔ XP & Level system to gamify productivity
✔ Daily task cards with subtasks
✔ Notes for each task and each day
✔ Progress dashboard with stats
✔ 30-day activity heatmap
✔ Day streak counter
✔ Confetti celebration and sound effects
✔ Export / Import JSON backup
✔ Fully responsive design for mobile & desktop

## 🧠 How It Works

The app stores all task data using browser LocalStorage.

This means:

* No login required
* All data stays on your device

### XP System

Action → XP Reward

Complete a Task → +10 XP
Complete a Day → +50 XP

XP contributes to your Level progression.
